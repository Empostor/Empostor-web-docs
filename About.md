# About

## Project Origin

The original [Impostor](https://github.com/Impostor/Impostor) project by [AeonLucid](https://github.com/AeonLucid) pioneered the community-driven open-source reimplementation of the Among Us server. However, after being taken over by [miniduikboot](https://github.com/miniduikboot) and [NikoCat233](https://github.com/NikoCat233), Impostor increasingly shifted towards a closed-source mentality — and its feature set remained too basic for real-world server hosting.

[NikoCat233](https://github.com/NikoCat233) later developed **Nmpostor**, a server built on top of Impostor, but priced it at **$27 USD per person**. For an open-source community, this was prohibitively expensive. The community was left without a feature-rich, truly open Among Us private server.

That's when [HayashiUme](https://github.com/HayashiUme) and [Fangkuai](https://github.com/FangkuaiYa) set out to build [Next-Impostor](https://github.com/BunchHanpiDev/Next-Impostor) — a more capable, open alternative. The project grew quickly, but accumulated so many bugs and architectural issues that development had to be halted.

[Fangkuai](https://github.com/FangkuaiYa) then started [Fast-Impostor](https://github.com/BunchHanpiDev/Fast-Impostor) as a clean-slate rewrite of an enhanced Impostor server. With different architectural beliefs, [HayashiUme](https://github.com/HayashiUme) set out to build **Empostor** ([github.com/Empostor/Empostor](https://github.com/Empostor/Empostor)) — and Fangkuai happily agreed to contribute alongside his own work on Fast-Impostor.

Together, our goal is to give the community what it always deserved: a **stable, feature-rich, plugin-extensible, and truly open-source** Among Us private server.
