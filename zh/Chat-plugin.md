# 聊天管理插件

::: tip 插件功能
此功能由 **Empostor.Plugin.Chat** 插件（`cn.hayashiume.chat`）提供。它不是核心服务器的组成部分。
:::

## 概述

聊天管理插件记录所有游戏内聊天消息，并强制执行可配置的消息长度限制。

## 功能

### 聊天记录

所有玩家聊天消息都会记录到服务器控制台，包含玩家名称、好友代码和游戏代码。

### 消息长度限制

防止聊天中出现过长的消息。对普通玩家和房主分别设置不同的限制。

**配置（`config/boot_chat.json`）：**

```json
{
  "playerMaxMessageLength": 300,
  "hostMaxMessageLength": 1200,
  "tooLongMessage": "你的消息太长了！"
}
```

| 设置 | 默认值 | 说明 |
|---|---|---|
| `playerMaxMessageLength` | 300 | 普通玩家每条消息最大字符数 |
| `hostMaxMessageLength` | 1200 | 房主每条消息最大字符数 |
| `tooLongMessage` | `"Your message is too long!"` | 超出限制时显示的消息 |

## 与内置聊天过滤器的对比

| 功能 | 聊天管理（插件） | 聊天过滤器（内置） |
|---|---|---|
| 消息长度限制 | 支持 | 不支持 |
| 聊天记录 | 支持 | 不支持 |
| 屏蔽词 | 不支持 | 支持 |
| 刷屏/频率限制 | 不支持 | 支持 |
| 拦截/仅记录模式 | 不支持 | 支持 |

两个系统互为补充，可以同时使用。
